<h2>Rhombus Generator</h2>

<p align="center">
<img src="Rhombus Generator C#.png" width="100%" height="100%" alt="C#" /></p>

<p align="justify">

I wrote this program to answer a request for help in the Open Bootcamp community in Discord.
I took up the challenge and in about 30 minutes I made this parametric diamond generator.
I thought of playing on the quantity of spaces and on the specular projection.

I was satisfied with the result achieved, even if there is no user input check (error handling of wrong data type)
Furthermore, you could choose the font used for the graphic representation of the rhombus.
</p>
